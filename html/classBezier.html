Detailed Description

Bezier class represents function interpolation method using third-order Bezier curves. Formula for this curve is {B}(t) = (1-t)^3{P}_0 + 3t(1-t)^2{P}_1 + 3t^2(1-t){P}_2 + t^3{P}_3, t [0,1]
